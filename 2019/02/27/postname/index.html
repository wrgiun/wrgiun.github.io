四、选择fcitx输入法

从系统设置里面点击“语言支持”，
从语言支持里面选择小企鹅输入法（fcitx），默认为ibus，搜狗输入要安装在fcitx输入法上面。
你可能会打不开“语言支持”，只需执行命令sudo apt-get install -f 即可修复错误，打开语音支持
